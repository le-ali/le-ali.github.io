What is the differences between Margin, Border, and Padding?

Box Model

In CSS, all the contents are represented by a rectangular box. This box is composed of 4 parts – content, padding, border and margin – which make up the box model. Content from text, to images, to buttons all live inside a box. Every box has three available properties: margin, border, and padding. They function differently but all together can be adjusted to control its appearance.

Border

Every box has a border. The border separates the edge of one box from another.

Margin

Margins sit outside the edge of the border. You can set the width of a margin to create a gap between the borders of two adjacent boxes.

Padding

Padding is the space between the border of a box and any content contained within.

Conclusion

The box model in CSS in one of the most important concepts that you can learn. Border, Padding and Margin have different funtionality but are equally as important as each other. Border is the outline of the object. Padding is the space between the object and the border. Margin is the space between one object and another.
