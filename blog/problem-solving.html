When I was blocked on a simple problem

When I was on the manipulating objects task, I was blocked on step 4, which is "add a key to an object and assign it a value." When I finished coding this step, I found that I couldn’t get the correct answer back. Instead of getting an object back, I only got a string back. I found out about this problem by using console.log. At first, I couldn’t figure out why and got a bit frustrated. I had to tell myself to just relax and decided to take my dogs out for a walk. While I was walking my dogs, I suddenly realised that instead of assigning it a value, I assigned it a string.

What I learned from this is that I need to read the question properly and really understand what it wants me to do. It said assign the VALUE, not the STRING.

The time I elegantly solved a problem

The concatMyArray from the building methods task was a pretty straightforward and relatively easy one for me. I did a google search, learned the syntax of .join() and solved this task. I used console.log to check the result. I felt pretty happy when I got it right and it didn’t take me long to solve.

What I learned is that you can always check the syntax or examples of different methods online (from W3Schools, MDN, or anywhere really!) If you can’t remember how to use it or even don’t know how to use it, they give you clear description, syntax and good examples.
