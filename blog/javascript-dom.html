Describe what the DOM is and an example of how you might interact with it

What is DOM

The DOM, or Document Object Model, is structured representation of HTML documents. When the browser loads a web page, it creates a model of the page in memory.The browser structure this model using a DOM tree. The DOM is called an object model because the model(the DOM tree) is made of objects.

How to interact with it

The DOM allows JavaScript to access HTML elements and styles to manipulate them, which includes change text, HTML attributes and even CSS styles.

Explain the difference between accessing date from arrays and objects

In JavaScript, arrays and objects are both used to store and organise data, but they have different characteristics and ways of accessing their data.

Arrays

An array is an ordered collection of elements. The elements in an array are indexed starting from zero. You can access array elements using their index value.

Objects

An objects is an unordered collection of Key-value pairs.Unlike arrays, objects don’t have an inherent order for their properties. You can access objects properties using their keys.

Explain what functions are and why they are helpful

what are functions?

In the most simple form, a function is simply a piece of code that we can reuse over and over again in our code.

Why they are helpful?

Functions let you group a series of statements together to perform a specific task. If different parts of a script repeat the same task, you can reuse the function, which enhance code reusability. They provide a structured approach to solving complex problems, making your code more efficient and easier to work with.
